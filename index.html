GitStats - git history statistics generator

About

GitStats is a statistics generator for git repositories. It examines the repository and produces some interesting statistics from the history of it. Currently only git repositories can be used, and the only output format is HTML.

Requirements

  • Git
  • Python
  • Gnuplot
